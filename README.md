# 配置文件

- PORT 端口号
- WEB_DIRECTORY_PREFIX 网页文件目录
- LOG_MAX_NUMBER 日志最大数量
- Error_Page 错误页面
- Error_Page.CODE 错误代码
- Error_Page.LOCATION 错误页面文件地址
