**简要描述：**

- 强制下线接口

**接收指令：**

- `forcedOffline`

**接收方式：**

- WebSocket

**返回代码说明**

| 代码   | 代码说明                      |
|:-----|:--------------------------|
| 0    | 成功                        |

**返回示例**

```json
{
  "cmd": "forcedOffline",
  "code": "0",
  "msg": "您的账号在别的客户端登录，您将被强制下线"
}
```

**返回参数说明**

| 参数名  | 类型     | 说明   |
|:-----|:-------|:-----|
| cmd  | string | 指令   |
| code | string | 返回代码 |
| msg  | string | 返回消息 |

**备注**

- 更多返回错误代码请看首页的错误代码描述
