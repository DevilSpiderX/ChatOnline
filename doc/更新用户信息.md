# 更新用户信息

**简要描述：**

- 更新用户信息接口

**请求URL：**

- ` http://xx.com/updateInformation `

**请求方式：**

- POST

**参数：**

| 参数名     | 必选  | 类型     | 说明            |
|:--------|:----|:-------|:--------------|
| own_uid | 是   | string | 自己的账号         |
| nick    | 否   | string | 要改的网名         |
| age     | 否   | int    | 要改的年龄         |
| gender  | 否   | string | 要改的性别 '男'或'女' |
| intro   | 否   | string | 要改的简介         |

**返回代码说明**

| 代码   | 代码说明            |
|:-----|:----------------|
| 0    | 成功              |
| 1    | 失败              |
| 2    | own_uid参数不存在    |
| 1001 | 没有权限            |
| 1002 | 不是自己的账号         |

**返回示例**

```json
{
  "code": "0",
  "msg": "更新成功"
}
```

```json
{
  "code": "1",
  "msg": "更新失败（数据库原因）"
}
```

**返回参数说明**

| 参数名    | 类型     | 说明     |
|:-------|:-------|:-------|
| code   | string | 返回代码   |
| msg    | string | 返回消息   |

**备注**

- 更多返回错误代码请看首页的错误代码描述
