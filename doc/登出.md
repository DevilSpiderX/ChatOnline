**简要描述：**

- 登出接口

**请求URL：**

- ` http://xx.com/logout `

**请求方式：**

- POST

**参数：**

| 参数名 | 必选  | 类型     | 说明  |
|:----|:----|:-------|:----|
| uid | 是   | string | 账号  |

**返回代码说明**

| 代码   | 代码说明     |
|:-----|:---------|
| 0    | 成功       |
| 1    | 还未登录     |
| 2    | uid参数不存在 |
| 1002 | 不是自己的账号  |

**返回示例**

```json
{
  "code": "0",
  "msg": "001登出成功"
}
```

```json
{
  "code": "1",
  "msg": "该用户未登录"
}
```

**返回参数说明**

| 参数名   | 类型     | 说明    |
|:------|:-------|:------|
| code  | string | 返回代码  |
| msg   | string | 返回消息  |

**备注**

- 更多返回错误代码请看首页的错误代码描述
